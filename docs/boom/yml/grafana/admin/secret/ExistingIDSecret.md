# ExistingIDSecret 
 

## Structure 
 

| Attribute    | Description                                                                           | Default | Collection  |
| ------------ | ------------------------------------------------------------------------------------- | ------- | ----------  |
| name         | Name of the Secret                                                                    |         |             |
| idKey        | Key in the secret which contains the ID                                               |         |             |
| secretKey    | Key in the secret which contains the secret                                           |         |             |
