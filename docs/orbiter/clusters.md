# Supported Cluster

- Kubernetes with kubeadm

# More Possible Use Cases

- Other cluster managers
  - Nomad
  - Mesos
- Databases
  - etcd
  - CockroachDB
- Message Brokers
  - Kafka
  - RabbitMQ
  - NATS

If you desire an explicit implementation, file an issue. Also, pull requests are welcome and appreciated.
